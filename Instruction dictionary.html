Инструкция Общий вид Описание
aaa aaa Выполняет ASCII-коррекцию после сложения (BCD arithmetic)
aad aad Выполняет ASCII-коррекцию после вычитания (BCD arithmetic)
aam aam Выполняет ASCII-коррекцию AX регистра после умножения (BCD arithmetic)
aas aas Выполняет ASCII-коррекцию AX регистра перед делением (BCD arithmetic)
adc adc <цель> <источник> Выполняет сложение с переносом (к результату добавляется значение флага CF)
add add <цель> <источник> Выполняет сложение двух чисел
and and <цель> <источник> Выполняет логическое "И"
bound bound <регистр с индексом> <память, где лежат границы> Выполняет проверку индекса на выход за границы массива
call call <адрес> Вызов процедуры
cdq cdq Преобразование двойного слова в счетверенное слово (начало будет в edx, конец - eax)
clc clc Сброс флажка переноса (CF)
cld cld Сброс флажка направления (DF)
cli cli Сброс флажка прерывания (IF)
cmc cmc Инвертирование флажка переноса
cmp cmp <операнд1> <операнд2> Сравнение двух операндов (== sub, устанавливает флаги, не изменяя операндов)
cmps cmps Сравнение цепочечных операндов
cwde cwde Преобразование слова в двойное слово (EAX, AX)
daa daa Выполняет ASCII-коррекцию AL регистра после сложения (BCD arithmetic)
das das Выполняет ASCII-коррекцию AL регистра после вычитания (BCD arithmetic)
dec dec <операнд> Декремент на 1
div div <операнд> Беззнаковое деление
enter enter <размер в байтах> <обычно 0> Образование стекового кадра для параметров процедуры
hlt hlt Останов (Привилегированная команда)
idiv idiv <операнд> Знаковое деление
imul imul <операнд> ЛИБО imul <операнд1> <операнд2> Знаковое умножение
in in <регистр al или eax> <константа или dx> Ввод из порта
inc inc <операнд> Инкремент на 1
ins ins Ввод из порта в цепочку
int int <число> Вызов процедуры прерывания
int3 int3 Вызов процедуры прерывания 3
into into Вызов процедуры прерывания 4, если OF=1
iret iret Возврат из прерывания
Команды условной передачи управления
Условный переход осуществляется,
если выполняется определенное условие, заданное флагами процессора
Инструкция Общий вид Условие
jb jb <адрес> Если меньше
jbe jbe <адрес> Если меньше или равно
jecxz jecxz <адрес> Если счетчик ECX равен нулю
jl jl <адрес> Если меньше
jle jle <адрес> Если меньше или равно
jmp jmp <адрес> Безусловный переход (т.е всегда)
jnb jnb <адрес> Если не меньше
jnbe jnbe <адрес> Если не меньше или равно
jnl jnl <адрес> Если не меньше
jnle jnle <адрес> Если не меньше или равно
jno jno <адрес> Если нет флага переполнения
jnp jnp <адрес> Если нет флага четности
jns jns <адрес> Если нет флага знака
jnz jnz <адрес> Если нет флага нуля
jo jo <адрес> Если есть флаг переполнения
jp jp <адрес> Если есть флаг четности
js js <адрес> Если есть флаг знака
jz jz <адрес> Если есть флаг нуля
lahf lahf Пересылка из регистра флажков в регистр AH
lds lds <регистр для смещения> <память> Загрузка полного указателя (для DS)
lea lea <цель> <источник> Загрузка эффективного адреса
leave leave Отмена действия команды ENTER перед выходом из процедуры
les les <регистр для смещения> <память> Загрузка полного указателя (для ES)
lods lods Загрузка цепочечного операнда в аккумулятор
Команды перехода для повторения цикла
Эти команды используются для управления циклом со счетчиком в регистре ECX
Инструкция Общий вид Условие
loop ECX != 0
loopnz ECX != 0 и ZF = 0
loopz ECX != 0 и ZF = 1
mov mov <цель> <источник> Пересылка данных
movs movs Пересылка в специальные регистры и из них
mul mul <операнд> ЛИБО mul <операнд1> <операнд2> Беззнаковое умножение
neg neg <операнд> Получение дополнительного кода (изменение знака числа)
nop nop Пустая операция
not not <операнд> Инвертирование
or or <операнд1> <операнд2> Логическое включающее "ИЛИ"
out in <константа или dx> <регистр al или eax> Вывод в порт
outs outs Вывод цепочки в порт
pop pop <цель> Извлечение слова из стека
popa popa Извлечение из стека всех общих регистров
popf popf Извлечение из стека в регистр флажков
push push <операнд> Засылка операнда в стек
pusha pusha Засылка в стек всех общих регистров
pushf pushf Засылка в стек регистра флажков
rcl rcl <цель> <смещение> Циклический сдвиг влево с участием флажка CF
rcr rcr <цель> <смещение> Циклический сдвиг вправо с участием флажка CF
ret ret ИЛИ ret <число байт или слов,
которое надо освободить в стеке
после извлечения из стека адреса для возврата> 		Возврат из процедуры
rol rol <цель> <смещение> Циклический сдвиг влево
ror ror <цель> <смещение> Циклический сдвиг вправо
sahf sahf Пересылка регистра AH в регистр флажков
sal sal <цель> <смещение> Арифметический сдвиг влево (умножение на 2)
salc salc Установка регистра AL по флажку CF
sar sar <цель> <смещение> Арифметический сдвиг вправо (знаковое деление на 2)
sbb sbb <уменьшаемое> <вычитаемое> Вычитание с заемом
scas scas Сравнение (сканирование) цепочечных данных
shl shl <цель> <смещение> Обычный сдвиг влево
shr shr <цель> <смещение> Обычный сдвиг вправо
stc stc Установка флажка переноса
std std Установка флажка направления
sti sti Установка флажка прерывания
stos stos Сохранение цепочечных данных
sub sub <уменьшаемое> <вычитаемое> Вычитание
test test <операнд1> <операнд2> Логическое сравнение (== and, устанавливает флаги, не изменяя операндов)
wait wait Ожидание
xchg xchg <операнд1> <операнд2> Обмен данными - регистр/регистр или регистр/память
xlat xlat Перекодирование байта AL по таблице (адрес таблицы в EBX)
xor xor <операнд1> <операнд2> Логичечское исключающее "ИЛИ"
