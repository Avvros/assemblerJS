Примеры

Конец программы

push 00h
call [00405058h]
Любая программа должна заканчиваться именно так.

Ввод строки

mov edi, 00402000h
call 403000h
В регистр edi записывается адрес, по которому нужно разместить введённую строку.
По адресу 403000h находится подпрограмма ввода из стандартного потока.
Рассчитывать на сохранение значений регистров не стоит. После работы подпрограммы в регистре eax будет записано количество введённых символов.
Если строка заканчивалась концом строки, то код(ы) конца строки тоже будет введены и посчитаны.

Вывод строки

mov esi, 00402000h
mov eax, 0000000dh
call 403100h
В регистр esi записывается адрес начала строки.
В регистр eax записывается длина строки.
По адресу 403100h находится подпрограмма вывода в стандартный поток. Рассчитывать на сохранение значений регистров не стоит.

Повторение строки

mov edi, 00402000h
call 403000h
mov esi, 00402000h
call 403100h
push 00h
call [00405058h]
Это пример программы целиком, которая вводит из стандартного потока ввода строку и выводит её же в стандартный поток вывода.

Повторение строки без конца строки

mov edi, 00402000h
call 403000h
sub eax, 2
mov esi, 00402000h
call 403100h
push 00h
call [00405058h]
Конец строки представлен двумя байтами. Эти два байта находятся в конце строки. Соответственно, перед выводом уменьшаем длину строки на 2.
